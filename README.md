# Insurance Website Analytics Dashboard

## Student Information

- Student ID: W2120491

## Project Overview

This project was developed for the 5DATA004C Data Science Project Lifecycle module.

The dashboard analyses aggregated website data from an insurance company's website and provides interactive visualisations to help understand customer behaviour, marketing performance, and insurance sales.

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly Express
- GitHub

## Features

- Interactive Marketing Channel Filter
- Interactive Device Category Filter
- KPI Cards
- Business Insights
- Interactive Charts
- Download Filtered Dataset
- View Raw Data

## Dataset

insurance.data.aggregated.csv

## Running the Project

```bash
streamlit run app.py
```
